如何建立成功的关系？要内方外圆。

内方，是指内心守住正直的底线：不刻意讨好，不曲意奉承，不欺骗隐瞒，做一个品格端正、堂堂正正的人。

外圆，则是指外在展现圆融的智慧。处事懂得变通，待人注重和睦，让自己与环境和谐相处。
这里的“圆”，绝非圆滑取巧，而是圆通（懂得沟通）、圆融（彼此包容）、圆满（做事周全）。
心中有原则，不必处处张扬，否则易显得格格不入；外在虽随和，却始终不偏离内心的准则。

以下是十点具体建议。

# 一表人才
第一印象往往影响人际交往的起点，虽然光凭第一印象就判断别人是不可取的，但是几乎所有的人都凭第一印象判断，所以我们要给别人一个好的第一印象。

一表人才靠的是内心的涵养而不是外边的装饰。人有一个表面还有一个里面，会看人的人不止看你的表面而且看你的里面。
有些人就是外面很温和，内心很刚强。一表人才是要靠内心的修养，慢慢的把他表露出来，人家会看你这个东西。
有的人长的丑但是你希望跟他在一起，就是因为他内心很美，你跟他在一起你感到很放心，不必提防他，给人一种很信任的感觉。

# 两套西装
两套西装，不是指让你买两套很合身的，很名贵的西装。

两就是多的意思，两套西装的意思是不同的场合要穿不同的西装，指衣着要适应不同场合，要随和不要太过标新立异。
大家很怕都穿一样的衣服，但是也很怕完全不一样，太过特殊的。因此不要太过标新立异，让大家对你有一种敬而远之的感觉。

# 三杯酒量
在社交中，酒是常见的沟通媒介。一个人不喝酒人际关系不一定好，太过喜欢喝酒人际关系一定坏。

酒可以拉进人与人之间的距离，但是酒后会乱性，适量的酒对人际关系有好处，对身体有好处，过量的就会伤害身体，破坏人际关系。

不要一坐上宴席就把酒杯倒过来说我不喝酒，这样你就引起他人的注意了，可能会拼命的灌你酒，是给自己找麻烦。
人家跟你倒酒，你就让他倒，人家举杯你就一起举杯，跟着打混没有人注意就没事。

三杯酒量的意思是要随和一点，不要在嗜好上面跟人家格格不入，不要排斥这些人与人之间交往的媒介。
三杯酒量是社交场合必要的修养，但是不要过量的饮酒，稍微喝一点酒场面就比较融洽，大家都比较有话说。

# 四圈麻将
“麻将”在此泛指各类社交活动。下象棋属于“麻将”，打网球也是属于“麻将”。
这个圈子的人喜欢做什么活动，我们为了跟他们打成一片，我们多少要有一点参与。

一个人是你的目标要接近的人，他喜欢钓鱼，最好的方法就是你也去钓鱼，你如果始终不钓鱼就跟他无法接近。
要认识一个人，找人去介绍不是办法，因为介绍一次两次他都对你没有印象，你自己去毛遂自荐，他会害怕，可以打听人家喜欢什么，打听到他喜欢钓鱼，赶快买一个钓鱼竿，你就跑去说你喜欢钓鱼，我也喜欢钓鱼，那就完了。不要那么着急，你在那个地方远远的钓，慢慢的靠近他，他问你干嘛，你就说你只喜欢钓鱼，如果你直接把名片给他，他就知道，搞了半天是要认识我。一定要让他感觉你的动机很纯正，他才不会防备你。
有一天他问你到底是干什么的，这个时候已经取得信任了，你说你是卖汽车的，放长线，钓大鱼。

# 五方交友
交友不宜自我设限。不同背景、不同行业的人，都可能带来新的视角与机会。
有些人一看兴趣不同然后就不来往，这样就没有什么朋友了，然后见识也窄小。你永远也不知道你什么时候需要什么样的朋友。

# 六出祁山
做事应有韧性。如同诸葛亮六出祁山，明知道不可为还要这么做，这是在告诉我们，虽然做不到，你也要去做，所以到现在人们依然很敬仰他。
这就告诉我们只问应该不应该，少问可能不可能，谁也不知道有没有可能。

# 七术打马
打马就是拍马屁，但是我们不要拍马屁，而是要制造很浓的马屁味道。
术就是要有一些不同的方式，技术，巧妙的法术。

我们要让他听起来很受用而事后想起来觉得你不是拍马屁，这样就是马屁味道。

老板讲话永远是对的，不要当面顶撞他，但是你要事后告诉他这是行不通的，他就不觉得你是个马屁精。
人前歌功颂德，事后是唯唯诺诺，那就是马屁精。要七成的马屁味道，三成的真诚劝告。

# 八口吹牛
一个人不吹牛，别人永远不知道你做了什么事情，一个人过分的吹嘘自己，那所有人都对你没有信任感。
所以，适当地表达自己、肯定自己是必要的，否则他人可能忽视你的能力与贡献。

一件事情你可以把他说的很平淡，也可以稍微的添油加醋。说别人的事情，稍微夸大一点就相当于夸大自己。
吹牛就是给同事一些面子，尽量把功劳让给他，他自然会让给你，功劳是让出来的，让来让去大家都有功劳，抢来抢去大家都没有功劳。
要人捧人，而不要人损人，人损人就是损自己，人捧人就是捧自己。我对你客气，你要对我客气，你对我不客气，我会对你更狠，中国人就是这种个性。

# 九分努力
任何关系的维系，最终仍离不开实实在在的付出。他人愿意信任你、支持你，往往是因为看到你的努力与担当。
人家帮助我们，我们更要争气，人家提携我们，我们更要表现的让人家有面子，所以要靠努力。

# 十分忍让
忍耐远比努力重要，一个人只要不能忍耐，前面的九个统统归于零。心好是没有人知道的，脾气不好是所有人都知道的。

如果把忍耐的事情当做坏事情，那早晚有一天会爆发，如果全当做好事情，那你心里会很感激他。
忍耐不是靠压抑的，而是要把他化掉，化到最后根本不需要什么忍耐。良药本来就苦口，好听的话会害你，难听的话对你才有帮助。

# 总结
尽量让大家都去接受你，不要让大家一看到你就害怕，害怕就跟你保持很大的距离，你就失去了机会。

三杯酒量是在各种不同交际场合里面，你能够跟别人有话讲，打成一片彼此融入。
四圈麻将，是指社交游戏都要参与一点，不一定要最好。
五方交友，是指不要给自己设限，三教九流朋友都有，你只要不跟他一起做坏事就好。
六出祁山就是要有无比的毅力，应该做事情不折不挠。
七术打马和八口吹牛，我们也不要排斥这些东西，我是君子，凡事小人行径都不做，也不必，不要嫉恶如仇。别人用的很好，我拿来用用有什么不好。

最后的九分努力和十分忍耐很关键，忍字是心上面插了一把刀，你还无动于衷，忍耐的功夫很难，它是上乘的功夫。
所有给你难看的、骂你的，你不要去怪他，一怪你就受气了，要感谢他而不是埋怨他，那你整体就不一样了。
